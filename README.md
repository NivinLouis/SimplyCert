# SimplyCert – Free Bulk Certificate Generator | Upload Your Design, Add Names from CSV

# Generate Bulk E-Certificates From Your Own Design — Free

**SimplyCert** is a free, browser-based bulk e-certificate generator that eliminates the manual work of personalizing certificates. Unlike other tools that force you into rigid templates, SimplyCert lets you **bring your own design** and handles the one tedious part: stamping every attendee's name onto it, perfectly, at scale.

Generate personalized e-certificates in bulk for free. Upload your own certificate design, paste a name list or upload CSV, mark the text position, and download individual PDFs. No signup. No data upload. 100% private.

---

## 🚀 The workflow takes under two minutes:

1.  **Upload your design** — PNG, JPG, or PDF, exactly as you designed it (in Canva, Photoshop, etc.).
2.  **Drop in your attendee list** — Paste names, upload a CSV, or type them manually.
3.  **Mark the name area** — Draw a rectangle directly on your certificate preview where the name should go.
4.  **Style & Preview** — Choose from 30+ Google Fonts, set size, color, alignment, and preview instantly.
5.  **Export** — Download every certificate as an individual PDF, all packaged in a single ZIP.

## 🔥 Key Differentiators

### 🎨 Bring Your Own Design
Don't be locked into someone else's templates. Whether you designed it in Canva, Photoshop, or hand-painted it, SimplyCert respects your creative work. It just adds the names.

### 🔒 100% Client-Side Privacy
No files are ever uploaded to a server. All rendering, PDF creation, and data processing happen **locally in your browser**. Your data (and your attendees' names) stays on your machine.

### ⚡ No Signup, No Watermarks, No Subscriptions
SimplyCert is a completely free tool. No account required. No intrusive watermarks on your professional certificates. Just open the tab and start generating.

### 🖱️ Visual "Draw-to-Position"
Avoid the frustration of pixel coordinates. Simply draw a box where the name goes, and the tool handles the math. It's built for organizers, not developers.

## 🛠️ Technology Stack

Built with a focus on performance and privacy using the latest web technologies:

*   **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
*   **Canvas Engine**: [Fabric.js](http://fabricjs.com/) for visual coordinate mapping.
*   **PDF Generation**: [pdf-lib](https://pdf-lib.js.org/) for high-speed client-side PDF creation.
*   **Parsing**: [PapaParse](https://www.papaparse.com/) for lightning-fast CSV handling.
*   **Compression**: [JSZip](https://stuk.github.io/jszip/) to bundle individual PDFs into one download.
*   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) for a premium, responsive UI.
*   **State Management**: [Zustand](https://zustand-demo.pmnd.rs/) for seamless cross-step configuration.

## 🛠️ Getting Started (For Developers)

If you'd like to run SimplyCert locally or contribute:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/SimplyCert.git
    cd SimplyCert
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
